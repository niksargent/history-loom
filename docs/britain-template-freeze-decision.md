# Britain Template Freeze Decision

## Decision

Britain is now accepted as `template dataset 1`.

It should move out of active expansion mode and into:
- selective validation
- occasional targeted correction if a concrete weakness appears
- reuse as the reference workflow for dataset 2

## Why this decision is justified

Britain now has:
- workable event density across the lens
- no critically isolated periods
- a usable echo network
- explicit validation boundaries
- a documented maturity review
- a documented close-out checklist
- a reusable authoring and review workflow

## What this does not mean

It does not mean Britain is perfect.

It means:
- Britain is good enough to stop blocking the next phase
- further Britain changes should be selective rather than open-ended
- the project can now shift active effort to dataset 2 preparation

## Immediate consequence

The next active planning task is:
- create the dataset 2 charter

The next active product task is:
- define the minimum mechanics needed for dataset-level switching
