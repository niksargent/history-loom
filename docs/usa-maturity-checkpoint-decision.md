# USA Maturity Checkpoint Decision

## Decision

The USA dataset is accepted as:

`template dataset 2 at usable-draft maturity`

It should not stay in open-ended enrichment.
It should move into:
- selective validation
- selective correction where a real weakness appears in use
- reuse as the baseline method for France chartering

## Why this decision is justified

The USA dataset now has:
- a stable dataset charter and lens choice
- working app integration
- usable event density across the field
- a workable echo network
- snapshots and lived voice across all periods
- a completed period-quality review
- explicit authoring standards derived from Britain and USA together

It does **not** yet have:
- fully validated draft additions
- cross-dataset-comparable maturity
- uniformly strong echo confidence across every period

That is acceptable.
The checkpoint is not saying USA is final.
It is saying USA is mature enough to stop blocking France.

## What this means

### USA should now be treated as

- a live, usable draft dataset
- stable enough for product use
- still subject to selective validation

### USA should not now be treated as

- a dataset that needs more broad content expansion by default
- a cross-dataset compare baseline
- a finished, fully trusted canonical field

## Remaining USA watchpoints

The most important watchpoints remain:
- `us-p01`
- `us-p02`
- `us-p07`
- exploratory or moderate-strength echoes
- comparatively light grounding in a few pressures, especially where event support is thin

These are now maintenance and validation issues, not reasons to delay France.

## Immediate consequence

The next active planning task can now shift to:
- France chartering

The next active USA task is:
- selective validation only where a concrete issue emerges

## Product consequence

The tool now has:
- Britain as template dataset 1
- USA as template dataset 2 at usable-draft maturity

That is enough to support a serious multi-dataset product direction without pretending that cross-dataset compare is ready yet.
