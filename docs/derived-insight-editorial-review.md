# Derived Insight Editorial Review

## Purpose

Review the current derived-insight output as an editorial layer rather than treating every strong computation as a public claim.

This review is about:
- public wording
- legibility
- historical honesty
- deciding which internal cross-dataset cousin candidates are worth holding for later release

## Public prompt review

### USA

The first USA pilot is now strong enough to stay public, but needed one clear correction:
- internal family names such as `mobilised settlement` were too academic or too compressed for public prompts

Public prompt rule now:
- family prompts should say what the pattern feels like, not what the internal clustering system calls it

Examples:
- `This era is one of the moments when order is rebuilt under pressure.`
- `This era is one of the moments when fast change starts to fray trust.`

Relationship prompts should also stay plain:
- `tends to rise with`
- `tends to pull away from`
- `often moves before`

That language is not analytically perfect, but it is much more readable in-product.
