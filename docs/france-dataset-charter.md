# France Dataset Charter

## Purpose

Define dataset 3 for The History Loom as France, using the Britain and USA standards baseline rather than the earlier ad hoc prototype method.

## Proposed dataset

- dataset name: `France, 1789-2025`
- short label: `France`
- scope: `France`
- intended first status: `seed`, moving to `usable`

## Why France

France is the strongest next dataset because it combines:
- very high public recognizability
- repeated regime change
- strong structural questions around legitimacy, revolution, republic, empire, reform, secularism, class conflict, and national identity
- obvious future comparison value against both Britain and the USA

It is a better next step than a looser Europe-wide field because:
- the political story is stronger and cleaner
- regime change is more legible to non-specialists
- the echoes are likely to be sharper and less diffuse

## Audience value

For a non-specialist audience, France should illuminate:
- revolution and refounding
- republic versus empire
- rights, citizenship, and secularism
- class conflict and reform
- occupation, resistance, and reconstruction
- postwar centralization, protest, and fragmentation

## Core historical questions

The dataset should help users explore:
- when political legitimacy is rebuilt and when it collapses
- how revolution changes what citizenship means
- when central state power stabilizes the field and when it provokes backlash
- how class, religion, identity, and republic interact
- how France differs from Britain and the USA when similar pressures appear

## Scope boundary

The first France version should begin in `1789`, not deep medieval or early-modern France.

Reason:
- the Revolution is the clearest public starting point
- it gives the product a strong narrative threshold
- it avoids spreading dataset 3 too thin before the method is fully proven

This does not claim that pre-1789 France is unimportant.
It is a deliberate v1 boundary.

## Planning assumption for the default lens

The working assumption is:
- one default civic lens
- comparable historical eras rather than equal-duration periods
- exact period boundaries to be finalized in the next step

France should not be forced into Britain's 12 equal periods or USA's 8-era structure unless that genuinely fits the historical field.

## Initial maturity target

France should aim for:
- `seed` first
- `usable` before any cross-dataset compare is exposed

It does not need to begin at `rich`.

## Pressure model assumption

France should begin by reusing the current shared pressure model unless a real gap appears during periodization and seeding.

The most likely high-salience pressures are:
- institutional legitimacy
- social cohesion
- moral certainty
- information acceleration
- inequality
- militarization
- public hope

## Standards inherited from Britain and USA

France should follow the shared authoring rules now documented in:
- `docs/dataset-authoring-workflow.md`
- `docs/dataset-authoring-standards.md`

That means:
- no event included by fame alone
- no echo included merely to thicken the graph
- lived voice required from the start
- explicit draft versus validated status maintained throughout
