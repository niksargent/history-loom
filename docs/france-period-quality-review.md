# France Period Quality Review

## Purpose

Execute `france-task-09` directly: a live-use clarity review over the current France guarded draft field.

This is a higher-reasoning step because the question is no longer whether France has enough files.
The question is whether the field is already legible, compelling, and structurally honest enough to support real use without immediately slipping back into open-ended enrichment.

## Review standard

Each period is judged on:
- event grounding
- pressure grounding
- echo usefulness
- snapshot readability
- lived-voice accessibility
- overall exhibit legibility

The goal is not to make every era equally dense.
The goal is to make every era usable, intelligible, and worth exploring.
